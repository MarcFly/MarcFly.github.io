Project List

Projects I have worked on, for more info on each one, go into their respective pages or ask me about it!

Qt Based Graphics Engine - 2020

Engine to visualize different shader postprocessing techniques in a Deferred Rendering Pipeline. Implmentation include single pass Depth of Field with Depth Bleeding, simple SSAO and Deferred Lighting. Engine base provided by Jesús Díaz García.

GLSL Initiation with Shadertoy - 2020

Raycasting Initiation for getting used to GLSL, done with Shadertoy.

Zurullito Painter - 2020

Simple shape painting tool done with QtCreator. Focus on learning QtCreator and understanding of the Signal/Slot system.

Malita: Soul of a Warrior - 2019

Bachelors Degree's 3rd Year class project, involving development of the engine using C++, OpenGL, dearImGui and other libraries. I was stationed as a generalist programmer in a team of 21 students.

DrunkEngine - 2018

Game Engine base with specific implementation of a CPU driven Skeletal Animation System. Usage of OpenGL (forward rendering pipeline), PCG, modified MathGeoLib, DevIL, ImGui, SDL, Assimp and Parson, between others. Project done along with Andreu Sacasas.
