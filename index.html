Qt Based Graphics Engine

Engine to visualize different shader postprocessing techniques in a Deferred Rednering Pipeline. Implmentation include single pass Depth of Field with Depth Bleeding, simple SSAO and Deferred Lighting. Engine base provided by Jesús Díaz García.

DrunkEngine

Game Engine base with specific implementation of a CPU driven Skeletal Animation System. Usage of OpenGL (forward rendering pipeline), PCG, modified MathGeoLib, DevIL, ImGui, SDL, Assimp and Parson, between others. Project done along with Andreu Sacasas.

Zurullito Painter

Simple shape painting tool done with QtCreator. Focus on learning QtCreator and understandfing of the Signal/Slot system.
